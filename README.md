# 🚁 DSA Drone Delivery System

A Java-based Drone Delivery System project built to demonstrate Data Structures and Algorithms concepts through a real-world simulation.

This project models drone delivery routes, weather conditions, charging stations, and no-fly zones — all managed efficiently using advanced algorithmic logic.

---

## 🧠 Overview

The DSA Drone Delivery System simulates how delivery drones optimize routes and resources while considering real-world constraints such as:

- Weather fluctuations
- No-fly zones
- Charging stations
- Delivery points and warehouses

This project was developed as part of coursework at Amrita Vishwa Vidyapeetham, showcasing practical applications of DSA concepts in logistics systems.

---

## 👥 Contributors

Developed collaboratively as a group academic project.

## ✨ Key Features

- ✅ Route Optimization using graph algorithms
- ✅ Segment Tree implementation for efficient range queries
- ✅ Weather condition simulation
- ✅ No-fly zone detection and handling
- ✅ Modular Java architecture using models, services, and utils packages
